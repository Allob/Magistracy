## Web Service

Сервис для просмотра новостей, редактирования источников (добавления, удаления, изменения) 

## API

# ресурсы:
 - коллекция новостей
 - новость
 - коллекция источников
 - источник

# запросы

 - GET /news/?source={ALL/id}&id={ALL/id}&limit={10/num}


 - GET /sources/


 - GET /sources/{id}


 - PUT /sources/{id}
