Сначала  исходный текст (file) делится на строки таблицы по символу конца строки. Далее для всех разделенных строк применяется функция map, которая формирует пары (target_verticle, weight) из 1ого и 2ого элементов строки таблицы, раздененной символом табуляции. 

Далее, ко всем таким парам применяется функция reduceByKey ( вместо groupByKey, которая принудительно вызывает shuffle перед групировкой), которая берет аккумулированное значение для ключа и прибавляет текущее значение для этого ключа. В итоге получаются сгрупированные по ключу пары (target_verticle, sum(weight))

Далее полученные результаты сохраняются в файл с добовлением разделителя \t
